# 钏钏选标器

一个面向**股票 / 黄金 / BTC / 山寨币**的单页投资分析工具，主打 **统一 K 线体验、自选组合跟踪、风险评估门控、热点与主线行情聚合**。

这是一个适合作品集展示的全栈小产品：前端是单文件网页应用，后端使用 FastAPI 提供行情、策略、自选、用户与风险评估接口，并通过 SQLite 持久化用户数据。

## 项目亮点

- **多资产统一分析界面**：股票、黄金、BTC、山寨币采用一致的查看与交互方式
- **统一多周期 K 线**：15 分、日线、周线、月线、季线、年线切换一致
- **用户系统**：支持注册、登录、会话管理
- **风险评估门控**：根据风险等级决定山寨币功能开放范围
- **自选组合**：支持添加、编辑、删除、查看组合收益
- **截图 / 文本识别导入**：可从截图或文字中提取股票/币种代码批量加入自选
- **市场信息聚合**：整合热点新闻、主线行情、市场概览与资金流说明

## 适合展示的能力

这个项目比较适合放进求职作品集，因为它同时体现了：

- **产品设计能力**：不是单一图表页，而是围绕“选标”场景组织完整流程
- **前后端整合能力**：前端页面、后端 API、数据库状态持久化打通
- **数据处理能力**：对股票、黄金、加密货币做统一字段封装与多周期处理
- **交互设计能力**：登录弹窗、风险评估、自选编辑、图表切换、OCR 导入等完整交互

## 本地运行

### 1. 创建并激活环境

```bash
cd chuansuan-selector
python3 -m venv venv
source venv/bin/activate
```

### 2. 安装依赖

```bash
pip install fastapi uvicorn aiosqlite pandas numpy httpx requests bcrypt pillow pytest
```

### 3. 启动项目

```bash
python server.py
```

默认运行在：

```text
http://localhost:8080
```
