# 钏钏选标器

一个面向 **股票 / 黄金 / BTC / 山寨币** 的多资产选标与跟踪工具，主打 **统一 K 线体验、自选组合管理、风险评估门控、热点与主线行情聚合**。


## 项目定位

如果把它用一句话概括：

> 一个把多资产分析、自选跟踪、风险评估和信息聚合放进同一套交互里的投资选标工具原型。


## 核心亮点

- **多资产统一体验**：股票、黄金、BTC、山寨币使用一致的分析路径
- **统一多周期 K 线**：15 分、日线、周线、月线、季线、年线切换逻辑统一
- **用户系统**：支持注册、登录、会话管理
- **风险评估门控**：不同风险等级对应不同功能开放范围
- **自选组合**：支持添加、编辑、删除、查看浮盈亏与阶段表现
- **OCR 导入**：可以从截图 / 文本中提取代码并批量加入自选
- **信息聚合**：把热点新闻、主线行情、市场概览整合到首页流中


### 1) 简介
- 看市场
- 选标的
- 分析走势
- 加入自选
- 记录持仓
- 根据风险等级控制可见功能

### 2) 前后端整合
- 前端负责交互、图表、页面组织、用户状态管理
- 后端负责 API、数据处理、登录鉴权、风险评估存储、自选数据持久化
- SQLite 负责轻量本地数据存储

### 3) 数据展示与统一抽象
- 把股票、黄金、BTC、山寨币这几类资产整理成统一展示方式
- 统一技术指标、周期切换、策略信号、分析结论输出格式

## 本地运行

### 1. 创建并激活环境

```bash
cd chuansuan-selector
python3 -m venv venv
source venv/bin/activate
```

### 2. 安装依赖

```bash
pip install fastapi uvicorn aiosqlite pandas numpy httpx requests bcrypt pillow pytest
```

### 3. 启动项目

```bash
python server.py
```

默认运行在：

```text
http://localhost:8080
```
